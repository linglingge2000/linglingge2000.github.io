Hamilton时间序列分析3——平稳ARMA过程

2022-05-01
Hamilton时间序列分析

一、平稳ARMA过程

[TOC]

1. Book：J.D. Hamilton, Time Series Analysis

2. 平稳性与遍历性定义

1‘ 平稳性

平稳性用于刻画时间序列的统计性质关于时间平移的不变性，保证许多统计量的渐进分布。

研究出发点是希望通过时间序列的历史数据，得到其未来的一些预测，即希望该时间序列在历史数据上的性质，在将来保持不变，这样的预测具有实际意义。

平稳性依照定义分为严平稳(也成为强平稳)和弱平稳(也称作协方差平稳或宽平稳)：

严平稳表示一个过程$\lbrace Y_t \rbrace$的任意 $j$ 期联合概率密度$f(Y_{t-j},\dots,Y_{t})$(即所有统计性质)仅与 $j$ 的取值有关，与时间t无关；

弱平稳表示一个过程$\lbrace Y_t \rbrace$的均值 $\mu_{t}$ 和 $t$ 阶自协方差 $\gamma_{jt}$ 都仅与间隔$j$有关，与时间 $t$ 无关: 即$E(Y_t)=\mu$, $E(Y_t-\mu)(Y_{t-j}-\mu)=\gamma_j$；

${\color{red}两者区别}$：

严平稳序列一阶和二阶矩不一定有限，若一个严平稳过程存在有限一二阶矩，则该过程一定是弱平稳过程。

若一个过程 $\lbrace Y_t \rbrace$ 的联合分布是多元高斯分布，则该过程既是严平稳过程也是弱平稳过程，称为$\color{red}高斯平稳过程$。

$\color{red}注意：下文所说的平稳过程若无特殊说明都表示弱平稳过程。$

2’ 遍历性

遍历性对于时间序列的意义类似于大数定律对于一般随机变量的意义。用于保证使用样本均值/样本自协方差来估计总体均值/总体自协方差的可行性。

对于平稳过程$\lbrace Y_t \rbrace$，若其自协方差 $\gamma_j$ 随着 $j$ 的增大，足够快地趋于0，则称该过程是遍历过程，即所有阶矩都具有遍历性。

特别地，若一个过程是${\color{red}高斯平稳过程}$，只要满足自协方差求和有限，即$\sum_{j=0}^{\infty}|\gamma_j|<\infty$，该过程所有阶矩都满足遍历性。

2. 几个常见过程

1‘ 白噪声过程

一个过程 $\lbrace\varepsilon_t\rbrace$ 满足 $(i)E(\varepsilon_t)=0, (ii)E(\varepsilon_t^2)=\sigma^2, (iii)E(\varepsilon_t\varepsilon_s)=0,$ for any $t \ne s$ ，称为白噪声过程。若不同时间独立，即上述$(ii)$改为$E(\varepsilon_t\varepsilon_s)=E(\varepsilon_t)(\varepsilon_s)$， 该过程称为独立白噪声过程。若在独立白噪声基础上添加条件$(iv)\varepsilon_t \sim N(0,\sigma^2)$，该过程称为高斯白噪声过程。

2’ 移动平均过程

$\pmb{一阶移动平均过程MA(1)}$：$Y_t=\mu + \varepsilon_t+\theta\varepsilon_{t-1}$，其中 $\mu,\theta$ 为常数，$\lbrace\varepsilon_t \rbrace$ 是白噪声过程。满足期望$E(Y_t)=\mu$，方差$\gamma_0=E(Y_t-\mu)^2=(1+\theta^2)\sigma^2$，一阶自协方差$\gamma_1=E(Y_t-\mu)(Y_{t-1}-\mu)=\theta\sigma^2$，高阶自协方差$\gamma_j=0$, for $j >1$。由于均值和协方差都与时间无关，该过程是宽平稳过程。

$\pmb{q阶移动平均过程MA(q)}$ ：$Y_t=\mu + \sum_{j=0}^{q}\theta_j\varepsilon_{t-j}$，其中$\mu,\theta_j$为常数，$\theta_0=1$。

$\pmb{无限阶移动平均过程MA(\infty)}$：$Y_t=\mu + \sum_{j=0}^{\infty}\theta_j\varepsilon_{t-j}$。

3‘ 自回归过程

$\pmb{一阶自回归过程AR(1)}$：$Y_t=c + \phi Y_{t-1}+\varepsilon_t$，其中$\lbrace\varepsilon_t \rbrace$ 是白噪声过程
